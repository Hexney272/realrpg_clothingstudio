Ide kerülnek majd a saját blank clothing template fájlok.

A runtime texture rendszerhez olyan streamed blank .ydd/.ytd ruha kell, amelynek textúra nevei egyeznek a config.lua Config.RuntimeTextures.slots értékeivel.

Példa:
txd = realrpg_blank_top_01
txn = blank_diffuse

A konkrét .ydd/.ytd elkészítése OpenIV/CodeWalker/Blender ruha pipeline feladat, ezt nem tudja tisztán Lua/NUI kóddal létrehozni a resource.
