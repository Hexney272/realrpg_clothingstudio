This folder is the manifest-driven blank garment pack area.

The included manifest.json is an EXAMPLE. It does not include real GTA clothing models.
To make the prints visible on the actual ped you must place real addon clothing files here:
- .ydd model files
- .ytd texture dictionaries
- clothing meta files where needed

The txd/txn values inside manifest.json must match the texture dictionary/name inside your .ytd.
The default texture name used by the runtime replacement is blank_diffuse.
