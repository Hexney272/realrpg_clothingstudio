Ide kerüljenek a valódi blank .ydd/.ytd garment fájlok.

Példa név:
realrpg_blank_top_01.ydd
realrpg_blank_top_01.ytd

A .ytd-ben a texture name legyen: blank_diffuse
A configban ehhez:
txd = 'realrpg_blank_top_01'
txn = 'blank_diffuse'

Valódi GTA clothing binary fájlt ez a build nem tartalmaz, csak a script oldali runtime rendszert és a guide-ot.
