# RealRPG Clothing Studio

RealRPG Clothing Studio is a premium FiveM clothing designer system built for roleplay servers that want custom player-created wearable designs.

Players can open a fullscreen studio, select garment templates, create designs with layers, upload images, use text/shapes/brush tools, generate AI artwork, save designs, print them as inventory items, wear them, publish them to a marketplace, and sell them to other players.

The system includes production-focused tools: admin approval, audit logging, health checks, runtime texture slot management, upload bridge support, cache maintenance, and marketplace payout handling.

Best suited for serious ESX/QBCore/Qbox servers using ox_inventory.
