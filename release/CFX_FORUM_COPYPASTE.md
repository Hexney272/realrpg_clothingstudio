# RealRPG Clothing Studio — Ingame Clothing Creator

Premium FiveM ingame clothing creator with a fullscreen studio UI, layer-based editor, image uploads, AI design bridge, runtime texture foundation, ox_inventory metadata items, marketplace, admin approval, audit logs, healthcheck and maintenance tools.

## Features

- Fullscreen clothing designer UI
- Layers, text, shapes, brush, image upload and paste
- My Designs library
- Printed clothing items with ox_inventory metadata
- Runtime texture foundation with slot management
- Garment pack manifest workflow
- Discord/CDN upload bridge
- Layer asset upload, no huge base64 design JSON required
- Gemini AI design bridge
- Marketplace / shop system
- Seller commission + offline payout queue
- Admin approval / reject / takedown
- Audit logs + optional Discord audit webhook
- Healthcheck and selftest commands
- Maintenance and cache cleanup tools

## Frameworks

- ESX Legacy
- QBCore
- Qbox foundation support
- ox_inventory integration

## Note

For real visible print-on-clothing, you need a streamed blank garment pack with matching `.ydd/.ytd` texture names. The script already includes the runtime and manifest workflow for that.
